# Research Fund Settlement Agent

GitHub Pages website for searching Korean research fund settlement guidance.

## Public Site

https://fhlyongko.github.io/research-fund-settlement-agent/

The site provides grounded quick answers for meeting expenses, expert consulting fees, lecture fees, overtime meal expenses, domestic and international conference travel, and equipment purchases.
