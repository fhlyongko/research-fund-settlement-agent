# LMS 공유용 안내문

## 바로 사용 링크

수업 LMS의 `외부 URL`, `학습자료`, `공지사항`, `HTML 블록`에 아래 링크를 연결하세요.

https://fhlyongko.github.io/research-fund-settlement-agent/

권장 제목:

`연구비 정산 검색 비서`

권장 설명:

`회의비, 출장여비, 학회등록비, 특강비, 자문료, 야근·특근 식대, 초과근무일지, 기자재·모니터 구매, 주유비 등 연구비 정산 기준을 검색할 수 있는 챗봇형 안내 페이지입니다.`

## LMS iframe 삽입 코드

LMS가 HTML 편집 또는 iframe 삽입을 허용하면 아래 코드를 붙여 넣으세요.

```html
<iframe
  src="https://fhlyongko.github.io/research-fund-settlement-agent/"
  title="연구비 정산 검색 비서"
  width="100%"
  height="780"
  style="border: 1px solid #dbe2ea; border-radius: 8px;"
  loading="lazy"
></iframe>
```

모바일 가독성을 조금 더 확보하려면 높이를 `900`으로 조정해도 됩니다.

## LMS 공지사항 문구

```text
연구비 정산 검색 비서 사용 안내

연구비 집행과 정산에 필요한 기준, 증빙, 메뉴 경로를 아래 링크에서 검색할 수 있습니다.

https://fhlyongko.github.io/research-fund-settlement-agent/

주요 검색 항목:
1. 회의비
2. 국내·해외 출장여비
3. 학회등록비
4. 특강비·자문료
5. 야근·특근 식대 및 초과근무일지
6. 연구재료비
7. 기자재·모니터 구매
8. 주유비/자가운전 유류비

주의사항:
이 페이지는 업로드된 지침과 매뉴얼 기반의 검색 보조 도구입니다. 실제 집행 전에는 과제별 협약 조건, 지원기관 지침, 산학협력단 최종 확인을 함께 진행하세요.
```

## 권장 LMS 등록 방식

1. LMS 강의실에 접속합니다.
2. `자료 추가`, `외부 URL`, `학습자료`, `페이지`, `공지사항` 중 하나를 선택합니다.
3. 제목을 `연구비 정산 검색 비서`로 입력합니다.
4. URL에 `https://fhlyongko.github.io/research-fund-settlement-agent/`를 입력합니다.
5. HTML 입력이 가능한 경우 iframe 코드를 붙여 넣습니다.
6. 새 창 열기 옵션이 있으면 켜는 것을 권장합니다.
7. 저장 후 학생/조교 계정에서 접속되는지 확인합니다.
